# Projection Start

# Set up the project

## Dependencies

To be able to run the project you need :

- [pnpm](https://pnpm.io/fr/installation)

## Installation

To run the project you need to run :

```bash
pnpm install
```

this will resolve all the deps.

## Dev

You can now start the project :

```bash
pnpm dev
```
